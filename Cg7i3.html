문제

반지름을 입력받아 원둘레와 원면적을 구하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
반지름을 입력하세요: 3
반지름 3.00인 원둘레는 18.85이고 원면적은 28.27입니다.
반지름 ( 3.00 + 1.00 )인 원둘레는 25.13이고 원면적은 50.27입니다. 


반지름을 입력하세요: 2
반지름 2.00인 원둘레는 12.57이고 원면적은 12.57입니다.
반지름 ( 2.00 + 1.00 )인 원둘레는 18.85이고 원면적은 28.27입니다. 


반지름을 입력하세요: 1
반지름 1.00인 원둘레는 6.28이고 원면적은 3.14입니다.
반지름 ( 1.00 + 1.00 )인 원둘레는 12.57이고 원면적은 12.57입니다.

알고리즘

프로그램 시작 
   반지름을 입력 
   먼저 반지름으로 원둘레 및 원면적 계산 
   (반지름+1)으로 원둘레 및 원면적 계산 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter17/Circle.c

	#include <stdio.h>

 

	// 매크로 상수 및 매크로 함수 정의

	#define PI 3.141592

1	#define CIRCUMFERENCE( r )  ( 2 * PI * ( r ) )

2	#define CIRCULARAREA( r ) ( PI * ( r ) * ( r ) )

	 

	// 프로그램 시작 

3	int main() {

		// 반지름을 입력 

		printf( "반지름을 입력하세요: " );

		double radius = 0.0;

4		scanf( "%lf", &radius );

 

		// 먼저 반지름으로 원둘레 및 원면적 계산 

5		double length =

6		                CIRCUMFERENCE( radius );

7		double area =

8		              CIRCULARAREA( radius );

9		printf( "반지름 %.2f인 원둘레는 %.2f이고 원면적은 %.2f입니다.\n", radius, length, area );

 

		// (반지름+1)으로 원둘레 및 원면적 계산 

10		length =

11		         CIRCUMFERENCE( radius + 1 );

12		area =

13		       CIRCULARAREA( radius + 1 );

14		printf( "반지름 ( %.2f + 1.00 )인 원둘레는 %.2f이고 원면적은 %.2f입니다.", radius, length, area);

	 

		// 프로그램 종료 

15		return 0;

	}
