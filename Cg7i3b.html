문제

반지름을 입력받아 원둘레와 원면적을 구하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
반지름을 입력하세요: 3
반지름 3.00인 원둘레는 18.85이고 원면적은 28.27입니다.
반지름 ( 3.00 + 1.00 )인 원둘레는 25.13이고 원면적은 50.27입니다. 


반지름을 입력하세요: 2
반지름 2.00인 원둘레는 12.57이고 원면적은 12.57입니다.
반지름 ( 2.00 + 1.00 )인 원둘레는 18.85이고 원면적은 28.27입니다. 


반지름을 입력하세요: 1
반지름 1.00인 원둘레는 6.28이고 원면적은 3.14입니다.
반지름 ( 1.00 + 1.00 )인 원둘레는 12.57이고 원면적은 12.57입니다.

코드 빈칸 채우기

# <> 
# PI 3.141592				 
# CIRCUMFERENCE r    2 * PI *  r  		 
# CIRCULARAREA r   PI *  r  *  r   		 
 
int main() { 
	printf( "반지름을 입력하세요: " ); 
	double radius = 0.0;						 
	scanf( "%lf", &radius ); 
	double length =  
	                CIRCUMFERENCE( radius ); 
	double area =  
	              CIRCULARAREA( radius ); 
	printf( "반지름 %.2f인 원둘레는 %.2f이고 원면적은 %.2f입니다.\n", radius, length, area ); 
	length =  
	         CIRCUMFERENCE( radius + 1 ); 
	area =  
	       CIRCULARAREA( radius + 1 ); 
	printf( "반지름 ( %.2f + 1.00 )인 원둘레는 %.2f이고 원면적은 %.2f입니다.", radius, length, area); 
 
	return 0; 
}
